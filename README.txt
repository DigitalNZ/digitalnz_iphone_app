DigitalNZ iPhone App 1.0

INSTRUCTIONS
1. Set up your iOS development environment (http://developer.apple.com/).
2. Download an API key from http://digitalnz.org/developer and set constApiKey on line 25 of SearchViewController.h to your api key value.
3. Download additional resources for this application from http://digitalnz.org/mobile if desired.
4. Find out more and get help at http://digitalnz.org/developer
